# 🎌 Anime Industry Analytics Dashboard

An interactive data analytics dashboard that explores trends, ratings, and growth of the anime industry from 1960 to 2024 using real MyAnimeList data.

## 🔗 Live Demo
[Click here to view the dashboard](https://anime-dashboard-6dryhappnq93dd5frapp643g.streamlit.app/) <!-- add your streamlit link after deployment -->

## 📊 Features
- **18,000+ anime** analyzed from MyAnimeList dataset
- **Anime releases per year** — industry growth trend
- **Top 10 highest rated anime** — interactive bar chart
- **Most popular genres** — genre distribution analysis
- **Score distribution** — rating histogram
- **Popularity vs Quality** — scatter plot analysis
- **AI-Generated Insights** — Groq LLaMA3 analyzes trends dynamically
- **Interactive filters** — filter by year range and minimum score

## 🛠️ Tech Stack
- **Python** — core programming
- **Streamlit** — web application framework
- **Pandas** — data cleaning and processing
- **Plotly** — interactive visualizations
- **Groq API (LLaMA3)** — AI-powered insights
- **MyAnimeList Dataset** — 18,000+ anime entries

## 🚀 Run Locally
```bash
git clone https://github.com/satyamsriramcharan/Anime-Dashboard.git
cd Anime-Dashboard
pip install -r requirements.txt
streamlit run app.py
```
